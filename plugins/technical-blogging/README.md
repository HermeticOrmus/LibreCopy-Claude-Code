# Technical Blogging Plugin

> Write engaging technical blog posts with proper structure, code examples, and SEO optimization.

## Purpose

Creates technical articles that teach, inform, and engage developer audiences. Handles the full spectrum from "deep dive into X" to "how we solved Y" to "announcing Z." Ensures posts are findable through search and shareable on social media.

## Agents

| Agent | Role |
|-------|------|
| `tech-blogger` | Writes engaging technical articles with narrative structure |
| `seo-optimizer` | Optimizes technical content for search engines and discoverability |

## Commands

| Command | Description |
|---------|-------------|
| `/blog-post` | Write a technical blog post on a specified topic |

## Skills

| Skill | Description |
|-------|-------------|
| `blog-structures` | Blog post structural patterns for different article types |

## Usage

```
/blog-post --topic "Building type-safe APIs with Hono and Zod"
/blog-post --topic "Our migration from REST to GraphQL" --type case-study
/blog-post --topic "What's new in v3.0" --type announcement
```

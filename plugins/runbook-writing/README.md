# Runbook Writing

Operational runbooks, incident response docs, SOPs

## What's Included

### Agents
- **Runbook Writer** - Specialized agent for Operational runbooks, incident response docs, SOPs

### Commands
- `/write-runbook` - Quick-access command for runbook-writing workflows

### Skills
- **Runbook Patterns** - Pattern library and knowledge base for runbook-writing

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/write-runbook analyze

# Use the command with specific input
/write-runbook generate --context "your project"

# Reference patterns from the skill
"Apply runbook-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for runbook-writing
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback

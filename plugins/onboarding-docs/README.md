# Onboarding Docs

New user onboarding, quick start guides, setup documentation

## What's Included

### Agents
- **Onboarding Writer** - Specialized agent for New user onboarding, quick start guides, setup documentation

### Commands
- `/doc-onboarding` - Quick-access command for onboarding-docs workflows

### Skills
- **Onboarding Patterns** - Pattern library and knowledge base for onboarding-docs

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/doc-onboarding analyze

# Use the command with specific input
/doc-onboarding generate --context "your project"

# Reference patterns from the skill
"Apply onboarding-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for onboarding-docs
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback

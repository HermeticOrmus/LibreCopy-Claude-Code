# README Engineering Plugin

> Craft compelling, well-structured README files that serve as the front door to any project.

## Purpose

A README is the single most important document in a repository. This plugin generates READMEs that balance marketing (why should I care?) with engineering (how do I use it?) to convert browsers into users and users into contributors.

## Agents

| Agent | Role |
|-------|------|
| `readme-architect` | Designs README structure, content hierarchy, and narrative flow |
| `badge-specialist` | Generates shields.io badges for build status, coverage, license, and more |

## Commands

| Command | Description |
|---------|-------------|
| `/readme` | Generate or overhaul a project README |

## Skills

| Skill | Description |
|-------|-------------|
| `readme-templates` | README structural patterns for different project types |

## Usage

```
/readme --type library
/readme --type cli-tool --badges ci,coverage,npm
/readme --type saas --sections features,quickstart,api,deploy
```

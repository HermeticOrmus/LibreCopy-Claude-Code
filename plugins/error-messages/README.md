# Error Messages

Error message writing, error codes, troubleshooting guides

## What's Included

### Agents
- **Error Message Writer** - Specialized agent for Error message writing, error codes, troubleshooting guides

### Commands
- `/doc-errors` - Quick-access command for error-messages workflows

### Skills
- **Error Message Patterns** - Pattern library and knowledge base for error-messages

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/doc-errors analyze

# Use the command with specific input
/doc-errors generate --context "your project"

# Reference patterns from the skill
"Apply error-message-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for error-messages
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback

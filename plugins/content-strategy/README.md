# Content Strategy

Documentation strategy, content planning, information architecture

## What's Included

### Agents
- **Content Strategist** - Specialized agent for Documentation strategy, content planning, information architecture

### Commands
- `/content-plan` - Quick-access command for content-strategy workflows

### Skills
- **Content Strategy Patterns** - Pattern library and knowledge base for content-strategy

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/content-plan analyze

# Use the command with specific input
/content-plan generate --context "your project"

# Reference patterns from the skill
"Apply content-strategy-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for content-strategy
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
